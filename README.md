# TODOアプリ

シンプルなタスク管理アプリです。

## 機能

- タスクの追加（Enterキーまたは＋ボタン）
- 完了チェック（チェックボックスをクリック）
- タスクの削除（ゴミ箱アイコン）
- 完了済みをまとめて削除

## 開発環境

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000) で確認できます。

## Vercelへのデプロイ

```bash
npm install -g vercel
vercel
```

または GitHub リポジトリを Vercel に連携するだけで自動デプロイされます。
